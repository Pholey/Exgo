Boilerplate API written in Go

## Getting started:
  You will need postgres as well as redis pre-installed with no special configurations

  Install Deps:
  `$ ./script/bootstrap`

  Provision database:
  `$ ./script/recycle`

  Test it:
  `$ ./script/test`

  Build it:
  `$ go build`

  Run it:
  `./Exgo`
